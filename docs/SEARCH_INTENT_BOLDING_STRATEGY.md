# Search Intent Bolding Strategy

Bold semantic emphasis is not decoration. It is a controlled signal system for users, search engines and AI retrieval systems.

## Rule

The generator may bold only exact phrases from the approved search-intent vocabulary when the phrase already exists in Sonia's copy or in approved hub descriptions.

The system must not bold random repeated words, generic words, or internal planning language.

## Why This Helps

The bold phrases help readers scan dense content and help crawlers identify the most important entity relationships inside long passages.

The value is not from bold text alone. The value comes from consistent alignment across:

- page title
- H1 and H2 structure
- body copy
- internal links
- semantic hubs
- JSON-LD
- sitemaps
- agent files

## QA Rule

Any new bold phrase must be added to the controlled vocabulary with:

- phrase
- topic
- intent
- reason
- rank value

If a phrase cannot be justified, it should not be bolded.
