# Iconography System

The site icon system exists to reduce cognitive load while keeping the visual language specific to Sonia McRorey's business: strategic image, executive presence, perception, leadership, professional identity and internal regulation.

## Rule

Icons must support user context. They must not introduce decorative, lifestyle, beauty, fashion influencer, or generic coaching meaning.

The generator uses one semantic icon map for:

- topic chips
- sentence insight cards
- service system orbit nodes

This keeps the meaning consistent across the entire static site without changing layout structure.

## Approved Icon Meanings

| Topic | Icon Concept | User Context |
| --- | --- | --- |
| Identidad | Professional identity card | The work starts with identity, values, role and self-concept. |
| Presencia | Centered executive posture | Presence is how the person occupies space and communicates authority. |
| Percepción | Eye with signal points | Image is interpreted by others through trust, coherence and credibility. |
| Decisión | Structured decision board | Image choices must become clear, intentional and usable. |
| Liderazgo | Executive growth chart | Sonia's work supports authority, leadership visibility and professional growth. |
| Empresa | Boardroom / organization case | The offer includes teams, brands, companies and organizational image. |
| Color | Professional swatch system | Color is treated as a strategic professional tool, not beauty styling. |
| Guardarropa | Hanger / wardrobe system | Wardrobe work is practical, coherent and functional. |
| Mentalidad | Internal regulation map | Presence is sustained by internal clarity, confidence and nervous-system regulation. |

## Implementation Boundary

Do not change section structures, copy, routes, headings, cards or layout just to add icons.

If an icon cannot be tied to Sonia's business intent, do not use it.
